# Valorant-Strat-Template
A template to create websites from obsidian valorant strats

Part of this can be used to make a website from any obsidian vault

The website will be created using tuftecss with a sidebar to help navigate

# Basis of the template 


## Header

Each .md file should have a header with 
`#{{Title}}`
`##Subtitle`

Then continue with the rest of the document

## File format

Each main file will have each main single hashtag header turn into a section (h1). any headers with more hastags will be turned into the according hx element

### Side notes

Sidenotes are a beautiful form of verbal communication

anything in `()`(brackets) will be turned into either a sidenote or a marginnote.
marginnotes are opened by clicking the + on mobile

### figures

Adding an image and text directly below will create a figure with the label being that text

### Sidebar

The sidebar will be created from the file structure of the root folder
